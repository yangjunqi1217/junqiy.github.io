概览 Overview

本仓库用于展示我们的视频幻觉评估基准，包括 忠实性幻觉（Fidelity Hallucination） 与 事实性幻觉（Factual Hallucination） 两个部分。

忠实性幻觉侧重 模型输出是否忠实于输入视频内容， 每个维度给出一个典型的 prompt + video 对，用于直观展示 benchmark 的设置。

事实性幻觉部分目前只预留结构，方便你后续根据实际数据集补充。

忠实性幻觉（Fidelity Hallucination）

忠实性部分按照四大类（Class1）和细粒度维度（Class2）组织： Entity Perception、 Action Perception、 Relational Reasoning、 Comprehensive Reasoning。 每个 Class1-Class2 对应一个示例。你可以在下方卡片中填入真实的 prompt 和视频路径。

事实性幻觉（Factual Hallucination）

这里预留给你后续的事实性幻觉 benchmark 展示。你可以完全复用忠实性部分的结构：

  1. 根据你的事实性分类框架，定义一个 factualData 数组。
  2. 仿照 renderFidelity() 写一个 renderFactual() 函数。
  3. 用本地视频路径替换每个示例中的 video 字段。
